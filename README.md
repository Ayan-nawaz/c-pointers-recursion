# C Pointers & Recursion Programs

This repository contains C programs that demonstrate the concepts of pointers, call by reference, recursion, and mathematical algorithms. These programs are useful for beginners who are learning C programming fundamentals and preparing for coding interviews or academic practicals.

## 📂 Programs Included

1. **swap.c** – Swap two numbers using a temporary variable.
2. **swapwithout3.c** – Swap two numbers without using a third variable.
3. **pointers.c** – Basic pointer operations and memory address demonstration.
4. **swapreference.c** – Swap two numbers using call by reference (pointers).
5. **hcf.c** – Find the Highest Common Factor (HCF) of two numbers.
6. **factorialrec.c** – Calculate the factorial of a number using recursion.

## 🎯 Topics Covered

* Pointers
* Pointer Variables
* Memory Addresses
* Call by Reference
* Recursion
* Function Calls
* HCF (Greatest Common Divisor)
* Number Swapping Techniques

## 💻 Technologies Used

* C Programming Language
* GCC Compiler
* Visual Studio Code

## ▶️ How to Run

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Compile the desired program using GCC.
4. Run the executable and provide the required input.

Example:

```bash
gcc pointers.c -o pointers
./pointers
```

## 📚 Learning Purpose

This repository is created for learning and practicing the core concepts of C programming, especially pointers and recursion. It is suitable for students, beginners, and anyone preparing for programming exams or interviews.
